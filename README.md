# Pawapuro-TextConverter
PS2初期の実況パワフルプロ野球におけるゲーム内文字コード（パワプロ文字コード）を、シフトJISと相互変換するツール

-----
# コマンド
`Pawapuro-TextConverter.exe [FILE] [OPTION]`

- コマンドライン引数無し
    - 対話モードでどの変換をするか選ぶ
        - シフトJIS→パワプロ文字コード
        - パワプロ文字コード→シフトJIS


**[FILE]** ファイルをバイナリモードで開いてシフトJISに変換する

**[OPTION]**

| OPTION | 説明 |
| ---- | ---- |
|**コマンド表示レベル**||
| -v0 | 改行、`[主人公]`のみ |
| -v1 | 通常レベル |
| -v2 | 詳細表示 |
|**その他**||
|-h<br>--help|ヘルプの表示|
